Welcome to Ludo Africa

The biggest online Ludo Community. New to the game? Follow quick steps for playing game. Also, please support Ludo Africa Project. Enjoy!!!

Create New Game? Steps

Select Player Mode. Choose 1 color if 4-Player or 2 colors if 2-Player. Click create game. Game code will be generated. Share game code with other player(s).

Join Existing Game? Steps

Click Join Game. Enter alpha-numeric game code. Click start game. Game will begin when game creator starts game. Enjoy!!!
